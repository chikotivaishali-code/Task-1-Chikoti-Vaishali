# 🤖 Rule-Based AI Chatbot

This project was developed as part of my Artificial Intelligence Internship at DecodeLabs. The chatbot is designed using rule-based logic in Python and responds to predefined user inputs using conditional statements and dictionary mapping.

The project focuses on understanding the foundational concepts of Artificial Intelligence before moving into advanced Machine Learning techniques.

---

# 📌 Project Overview

The Rule-Based AI Chatbot simulates basic human interaction by responding to user queries with predefined answers. It continuously accepts user input until the exit command is given.

This project demonstrates:
- Control flow in Python
- Decision-making logic
- Input processing
- Chatbot fundamentals
- Rule-based AI systems

---

# 🎯 Objectives

- Create a chatbot using Python
